# FuKōbō (工房)

> Professional painting for everyone.

FuKōbō is a free, cross-platform digital painting application built with **Electron**, **React**, and **TypeScript**, powered by the **Fuderu** brush engine.

Rather than being a standalone painting engine, FuKōbō serves as the reference desktop application for Fuderu, showcasing its capabilities in a complete creative workflow. The application is being designed around a modern, flexible workspace with a strong focus on performance, simplicity, and extensibility.

Every painting feature—from brushes and layers to future editing tools—is built on top of **Fuderu**, allowing the engine and the application to evolve independently while sharing the same core technology.

## Status

🚧 **Under Development**

FuKōbō is currently in active development. The initial focus is on building the application's workspace and user interface before integrating the complete painting workflow powered by Fuderu.

## About the Name

**工房 (Kōbō)** means *workshop* or *studio* in Japanese.

**FuKōbō** combines:

- **Fu** — from **Fuderu**
- **Kōbō** — workshop

Together it represents **"The workshop powered by Fuderu."**
